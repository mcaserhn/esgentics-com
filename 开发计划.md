# ESGentics.com 网站开发计划

## 项目概述

根据设计文档，ESGentics.com 是一个智能建筑技术公司网站，专注于 AI 驱动的建筑解决方案，总部位于新加坡，面向全球市场（严格排除中国大陆）。

## 技术栈选择

### 前端技术
- **框架**: React + Next.js 14
  - 优势：服务端渲染、路由系统、静态生成能力
- **样式方案**: Tailwind CSS
  - 优势：快速构建、响应式设计、与设计文档中的 CSS 变量兼容性好
- **UI 组件**: 自定义组件 + React Icons
- **动画效果**: Framer Motion
- **表单处理**: React Hook Form
- **验证**: Zod

### 后端技术
- **API 服务**: Next.js API Routes
- **表单提交**: 电子邮件发送 + 数据存储（可选）
- **部署**: Vercel 或 AWS Amplify

## 开发计划

### 阶段一：项目初始化与基础设置（1天）
1. **初始化 Next.js 项目**
   - 使用 `create-next-app` 创建项目
   - 配置 TypeScript
   - 安装必要依赖

2. **设置基础样式**
   - 实现设计文档中的 CSS 变量
   - 配置 Tailwind CSS
   - 导入 Catamaran 字体

3. **创建全局布局**
   - 实现 Header 组件
   - 实现 Footer 组件
   - 配置路由

### 阶段二：核心页面开发（5天）

#### 首页 (Home) - 1天
- 实现 Hero 区域（深蓝背景 + 3D 建筑线框动画）
- 实现价值主张区域
- 实现信任区域
- 添加 NVIDIA 合作伙伴标识

#### 技术引擎 (Technology) - 1天

- 实现深色板块设计
- 为技术部分添加可视化元素，如架构图、流程图等

#### 解决方案 (Solutions) - 1天
- 实现所有行业解决方案罗列在一页
- 为解决方案部分添加行业特定的案例和数据

#### 客户成功案例 (Case Studies) - 0.5天

#### 全球合规 (Compliance) - 0.5天

#### 关于我们 (About) - 0.5天

#### 联系我们 (Contact) - 0.5天

## 特殊要求实现

### 1. 中国大陆内容排除
- 全站搜索并确保无中国大陆相关词汇
- 地图组件排除中国大陆显示
- 数据主权声明明确无中国大陆数据存储

### 2. NVIDIA 品牌合规
- 正确使用 NVIDIA Logo
- 添加 "Powered by NVIDIA" 或 "NVIDIA Partner" 说明
- 遵循官方品牌指南

### 3. 数据主权
- 在页脚添加 "No data residency in mainland China" 声明
- 在合规页面详细说明数据存储位置
- 所有表单添加 Privacy Policy 链接和同意复选框

### 4. 响应式设计
- 实现设计文档中指定的断点
- 确保移动端体验良好
- 优化不同设备的布局
