## 🎯 批量生成技巧

### 使用 X/Y/Z Plot 批量测试

```
在 WebUI 中启用 "X/Y/Z plot" 扩展：

X轴: CFG Scale (7, 8, 9)
Y轴: Steps (30, 35, 40)

一次性生成 9 张变体，选择最佳参数组合
```

### 使用 Prompt Matrix

```
在 WebUI 中切换到 "Prompt matrix" 标签：

基础 Prompt: "futuristic smart building"
变量 1: "tropical / desert / European"
变量 2: "daytime / sunset / overcast"

自动生成所有组合，快速找到最佳方向
```

## 🔧 微调建议

### 如果图片质量不够

```
1. 增加 Steps: 30 → 50
2. 提高 CFG: 7 → 9
3. 启用 Highres. fix
4. 使用更好的 Upscaler: R-ESRGAN 4x+ Anime6B
```

### 如果品牌色不准确

```
1. 在 Prompt 中强调色值:
   "bright red #ed4f18" → "vibrant coral red #ed4f18, highly saturated"
   
2. 生成后用 Photoshop 调整:
   - 图像 → 调整 → 可选颜色
   - 红色 → 调整青色/洋红/黄色
   - 匹配 #ed4f18
```

### 如果出现中国元素

```
1. 加强负面 Prompt:
   添加: "Chinese style, Oriental, Asian traditional"
   
2. 使用 Negative Embedding:
   下载 "EasyNegative" embedding
   
3. 多次生成，挑选最佳
```
